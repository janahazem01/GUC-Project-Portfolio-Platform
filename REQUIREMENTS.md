# Updated Requirements

- Once a final thesis draft is selected, all other thesis drafts for that Bachelor Project must automatically become private and must not be visible to anyone else, including the assigned course instructor(s). Only the selected final draft remains public and visible to all permitted viewers.
- As a student, I should be able to send and cancel/unsend invitations to selected collaborators and/or course instructor(s) to join my project. Instructor invitations must be limited to the course instructor(s) assigned to that project's course.
- As a student, I should be able to view a list of my added collaborators and/or course instructor(s), including each invitation status: accepted, rejected, or no reply.
- As the project creator, I should be able to remove a collaborator from my project.
- As the project creator, I should be able to create, view, edit/update status, and delete/reorder project tasks with a short one-line description, assigned collaborator, status (pending, postponed, completed), and deadline. Collaborators can only change the status of tasks assigned to them. For Bachelor Projects, there are no collaborators; only the project creator can manage tasks.
- As a student, I should be able to view the full task list for a project, including all task details entered by the project creator.
- As the project creator, I should be able to reorder the tasks in my project according to importance.
- As a student, administrator, employer, or course instructor, I should be able to view a list of all my notifications.
- As a student, administrator, employer, or course instructor, I should be able to mark my notifications as read or unread.
- As a course instructor, I should be able to add, edit, and remove comments or feedback on each project task.
- As a course instructor, I should be able to add, edit, and remove comments or feedback on the project in general, including individual thesis drafts.
- As a course instructor, I should be able to rate the entire project out of 5, and that rating should be visible to everyone.
- As a student, I should be able to view comments left by course instructor(s) on the project in general, thesis drafts, and project tasks. These comments are only visible to the project creator, accepted collaborators, and assigned/accepted course instructor(s).
